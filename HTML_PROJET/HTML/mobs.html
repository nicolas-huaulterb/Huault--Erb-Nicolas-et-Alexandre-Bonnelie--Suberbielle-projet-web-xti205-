Qu'est-ce qu'un mob ?

Dans Minecraft, un mob est une créature vivante qui apparaît dans le monde. Les mobs peuvent être passifs (ils ne sont pas dangereux), neutres (ils attaquent seulement si on les provoque), ou hostiles (ils attaquent le joueur).

On trouve des mobs un peu partout : dans les plaines, les forêts, les grottes, les océans et même dans le Nether ou l'End.

Les différents types de mobs

Type de mob Description Exemples Où on les trouve ?
Passif Ils n’attaquent jamais le joueur. Vache, Mouton, Poule Plaine, forêt, montagne
Neutre Ils attaquent seulement si on les frappe ou si on les provoque. Enderman, Loup, Araignée Nuit, grottes, End
Hostile Ils attaquent automatiquement le joueur. Zombie, Squelette, Creeper Nuit, grottes, donjons
Boss Très puissants, difficiles à battre. Wither, Ender Dragon Nether / End
